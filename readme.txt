This plugin allows you to filter pages by parent in the admin dashboard.

== Description ==

This plugin displays a drop down menu in the admin dashboard that allows you to filter the page list by their parent page. No configuration is necessary.

== Installation ==

1. Upload the `custom-parent-page-filter` directory to `/wp-content/plugins/`.
2. Activate the plugin through the 'Plugins' menu in WordPress
